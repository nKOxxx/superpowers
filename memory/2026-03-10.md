## Late Night: Cognexia Smart Activation System

**Nikola's requirement:** Cognexia should proactively activate when:
- User types "new project"
- Project name is mentioned
- Smart trigger words detected
- "Precautionary activation" - save relevant data automatically

**Implemented:**

### 1. Smart Hook (`cognexia-smart-hook.js`)
- Analyzes every message for importance (0-10 score)
- Auto-stores high scores (>=7)
- Suggests storage for medium scores (5-6)
- Ignores low scores (casual chat)

### 2. Trigger Detection
**Explicit triggers:** "new project", "create project", etc.
**Smart triggers:**
- Decisions (7 pts): "I decided to..."
- Milestones (8 pts): "shipped", "v1.0"
- Issues (7 pts): "bug:", "error:"
- Security (9 pts): "api key", "password"
- Goals (6 pts): "goal is...", "plan to"
- Learnings (5 pts): "learned that..."
- Preferences (4 pts): "I prefer..."

### 3. Project Detection
- Extracts project name from "new project X"
- Detects mentions of known projects
- Auto-switches context
- Creates new projects automatically

### 4. Smart Storage Rules
```
Score 8-10: Auto-store + notify "💾 Saved..."
Score 5-7:  Auto-store silently
Score <5:   Ignore
```
