# QA Skill

Intelligent testing with smart test selection, coverage analysis, and systematic QA workflows.

## Commands

### Run changed tests only
```bash
superpowers qa --changed
```

### Full test suite
```bash
superpowers qa --full --coverage
```

### Security audit
```bash
superpowers qa --security
```

### Coverage analysis
```bash
superpowers qa --coverage --threshold 80
```
