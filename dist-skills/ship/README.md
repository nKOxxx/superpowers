# Ship Skill

One-command release pipeline for version bumps, changelog generation, and GitHub releases.

## Commands

### Ship minor version
```bash
superpowers ship minor
```

### Ship with push and release
```bash
superpowers ship patch --push --release
```

### Dry run
```bash
superpowers ship minor --dry-run
```

### With changelog
```bash
superpowers ship minor --changelog
```
