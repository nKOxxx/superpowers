# OpenClaw Superpowers

A curated collection of 4 essential skills that give your development workflow superpowers.

## 🌐 Browse - Browser Automation

Visual testing, accessibility audits, and visual regression with Playwright.

```bash
cd browse
npm install
npm run build
```

## 🧪 QA - Systematic Testing

Smart test selection, coverage analysis, and multi-framework support.

```bash
cd qa
npm install
npm run build
```

## 🚀 Ship - Release Pipeline

Semantic versioning, changelog generation, and publishing automation.

```bash
cd ship
npm install
npm run build
```

## 🎯 Plan CEO Review - Product Strategy

BAT framework and 10-star methodology for strategic decisions.

```bash
cd plan-ceo-review
npm install
npm run build
```

## Installation

```bash
# Install all skills
npm install
npm run build
```

## Usage with OpenClaw

Once installed, use the skills via OpenClaw commands:

- `/browse --url https://example.com --screenshots`
- `/qa --coverage`
- `/ship --bump patch`
- `/plan-ceo-review "Build vs buy auth"`
